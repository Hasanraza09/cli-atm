# CLI ATM Application

This is a command-line interface (CLI) ATM application built using Inquirer.js, Node.js, and TypeScript.

## Features

- Account Balance Inquiry
- Withdrawal
- PIN Verification
- FastCash feature

## Requirements

To run this application locally, you need the following:

- Node.js
- npm 
- TypeScript 

## Here is my npx command:
npx hraza09-cli-atm
